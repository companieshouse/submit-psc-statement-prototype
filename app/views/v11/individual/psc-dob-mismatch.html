Where to find the Companies House personal code

If you verified your identity using GOV One Login, you can find your code in your Companies House account. You’ll need to sign in (or create an account) using the same email address you used when verifying your identity.

If an authorised agent verified your identity, we sent your code to the email address they provided for you.

Check if the PSC's date of birth on the Companies House register is correct

To check the full date of birth we hold for you, someone authorised to file for the company will need to sign in to our WebFiling service.

If we have the wrong date of birth, they’ll need to submit a filing to correct it. Once it’s been corrected, you can return to this service and try again.

Correcting a date of birth

If you were added as a PSC as part of the company’s registration with Companies House, your date of birth can be corrected using the ‘Replace a document not meeting requirements (RP01)’ paper form.

If you were added as a PSC after the company was registered with us, it can be corrected using our WebFiling service.
